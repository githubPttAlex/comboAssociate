Getting Started

The bare minimum for using on a web page is linking to the library along with one of the styles and calling

                            
    <script src="../path/to/js/jquery-3.3.1.min.js"></script>
    <script src="../path/to/js/jquery-comboAssociate-v1-0.js"></script>

Basic Initialization

The bare minimum for using on a web page is linking to the library along with one of the styles and calling This will find and code inside of tags; it tries to detect the language automatically. If automatic detection doesn’t work for you, you can specify the language in the class attribute:

                              	
    $(document).ready(function(){
        $("#selectCombo1").comboAssociate();
    }); 
                            
                        
                              	
    <label for="selectCombo1">First Combo</label>
    <select id="selectCombo1" class="form-control comboAssociate">
          <option value="" selected>Choose...</option>
    </select>

Custom Initialization

The bare minimum for using on a web page is linking to the library along with one of the styles and calling This will find and code inside of tags; it tries to detect the language automatically. If automatic detection doesn’t work for you, you can specify the language in the class attribute:

                              	
    $(document).ready(function(){
        /* Create options array */
        var comboAssociateOptions = {
            url: 'ajaxBasicExample.php' /* PHP-URL where we get the data */
        }:
        
        /* Call comboAssociate plugin with options array */
        $(".comboAssociate").comboAssociate(comboAssociateOptions); 
    }); 
                            
                        
                              	
    <label for="selectCombo1">First Combo</label>
    <select id="selectCombo1" class="form-control comboAssociate">
          <option value="" selected>Choose...</option>
    </select>

comboAssociate Options

The bare minimum for using on a web page is linking to the library along with one of the styles and calling This will find and code inside of tags; it tries to detect the language automatically. If automatic detection doesn’t work for you, you can specify the language in the class attribute:

debug Option to determine if you want to show debug lines on console - by default is false
url PHP-URL where we get the data
chooseOption Option to determine if the combo use an 
<option>
tag for "CHOOSE AN OPTION" - by default is false
chooseOptionValue Option to determine the text of 
<option>
tag for "CHOOSE AN OPTION" - by default is 'Choose an option...'
chooseOptionText Option to determine the value of 
<option>
tag for "CHOOSE AN OPTION" - by default is '0'
empty Option to determine if you want we remove all the 
<option>
tags before populating again with server data - by default is true
dataType Option to determine dataType on ajax call - by default is 'json'
type Option to determine request type on ajax call - by default is 'POST'
onChange Function to make actions at onChange....
beforeFunction Function to make actions before....
afterFunction Function to make actions after....
